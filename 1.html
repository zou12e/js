变量

  • 原始值类型

    1. string

      • 例子："ABC"、"你好"

    2. number

      • 例子：1、100、NaN

    3. boolean

      • 例子：true、false

    4. null

      • 例子：null

    5. undefined

      • 例子：var a; a //undefined

    存储位置：

    栈（在栈中存储值，先进后出）

  • 引用值类型

    1. array

      • 例子：[1, 2, 3]、["你", "好"]

    2. function

      • 例子：function fun() {}

    3. object

      • 例子：{ name : "Jeff" }

    4. else

      • 例子：new Date()【日期】、/\d/【正则】

    存储位置：

    堆（堆中存值，在栈里面占一个位置,存储堆中的一个引用）
