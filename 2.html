运算

  • 运算符

    1. +

      • 数学运算
      • 字符串连接
        • 1 + 1 = 2 
"a" + 1 = "a1";

    2. -、 *、 /、 %

      • 数学运算
        •  0 / 0  NaN
 1 / 0  Infinity
-1 / 0  -Infinty

    3. ++、 --、 +=、 -=、 /=、 %=

      • 数学运算
        • a++    a=a+1
a--    a=a-1
a+=2   a=a+2
a-=2   a=a-2
a*=2   a=a*2
a/=2   a=a/2

    优先级：=最弱 、 ()最高

  • 比较运算符

    1. >、 <、 ==、 >=、 <=、 !=

      • 比较结果
        • null == undefined   true
NaN  == NaN         false

    比较结果返回真假

  • 逻辑运算符

    1. &&、 ||、 !

      • 逻辑计算
        • var a = 1 && 2;  // a 2
var b = 0 || 1;  // b 1
var c = 2 || 3   // b 2
